Notes for CSS

To start writing CSS we must open a style element in the head element section. CSS properties are written there.

To target all HTML categories, simply put the HTML tag without the angle brackets < > in the style section, following by squiggly brackets { } and within the squiggly brackets you will add the commands

To target a specific section, add an ID to the HTML tag (such as p id="" for a paragraph) and in the style section, you can set the CSS coding to that particular id by using the hashtag # to call the ID such as the heading of this paragraph (ID's are case sensitive, and must be unique, and therefore can only be used once). ID's can be very useful for linking with in a page

To format a group of things one can create a class. Classes are created in the style section using a "." and are created just like ID's only "class" is written instead. For example, the first points of each list is formatted using a class.
